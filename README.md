## 📍 Sobre o Projeto

O **NexuSaldo** foi desenvolvido com o objetivo de criar uma interface intuitiva e moderna para um sistema de controle financeiro pessoal. O foco principal foi a organização de elementos em tela, tipografia e o uso de cores que transmitissem confiança e clareza para o usuário final.

## 🎨 Destaques do CSS

Neste projeto, optei por utilizar técnicas modernas de CSS para facilitar a manutenção e escalabilidade do design:

### 🚀 Uso de `:root` (Variáveis CSS)

Utilizei a pseudo-classe `:root` para gerenciar a paleta de cores de forma centralizada. Isso permite que o tema do site seja alterado em um único lugar, afetando toda a aplicação.

```css
:root {
	--bg: #0F172A;
	--fundo-nav: rgba(30, 41, 59, 0.9);
	--card: #1E293B;
	--text: #E2E8F0;
	--text-btn: #00D1B2;
	--accent: #28a745;
	--accent2: #38BDF8;
	--border-main: rgba(0, 209, 178, 0.2);
}

```

### 📱 Layout Responsivo

O projeto utiliza propriedades de **Flexbox** para garantir que a interface se adapte a diferentes resoluções de tela, priorizando a experiência do usuário.
